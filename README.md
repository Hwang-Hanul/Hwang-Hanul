# Security Engineer Portfolio

실습 환경을 직접 구축하고,  
공격과 방어 흐름을 검증해온 보안 엔지니어 지망생입니다.

---

## What I Focus On
- Network / Web 기반 모의해킹 시나리오 설계
- 취약점 분석을 통한 공격 흐름 검증
- Python을 활용한 보안 테스트 자동화
- 로그 기반 공격 탐지 및 분석

---

## Featured Project
### Red Team – Network & Web Penetration Test Lab
- 역할: Red Team (Scenario Design & Attack Validation)
- 환경: Kali Linux, Ubuntu, Rocky Linux
- 내용:
  - 내부/외부 공격 시나리오 3종 설계
  - 네트워크, 웹, DB 취약점 연계 침투 검증
  - 공격 흐름 기반 보안 점검 체크리스트 작성

---

## Skills
- **Nmap / NSE**: 서비스 탐색 및 공격 표면 식별
- **Burp Suite**: 인증 흐름 분석 및 취약점 테스트
- **SQL Injection / XSS**: 정보 유출 가능성 검증
- **Python (requests)**: 인증 후 공격 자동화
- **Linux**: 서버 설정 분석 및 로그 점검

모의해킹과 취약점 진단을 통해
